## 📖 About this application

Anubis Firewall protects your Kubernetes environments. It uses machine learning to watch your network traffic. The tool spots threats before they reach your infrastructure. It stops unauthorized access and blocks malicious data requests. You get a set of tools to monitor and secure your system without deep coding knowledge. Anubis runs behind your services. It acts as a shield between the internet and your cluster.
